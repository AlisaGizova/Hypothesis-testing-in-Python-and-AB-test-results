# Проверка гипотезы в приложении для чтения и анализ результатов A/B-тестирования

**Задачи проекта:** Используя данные об активности пользователей из Москвы и Санкт-Петербурга в приложении для чтения, проверить гипотезу о различии среднего времени, проводимого в приложении, и проанализировать результаты A/B-теста нового интерфейса интернет-магазина для оценки его влияния на конверсию.

**Описание проекта:** Проведена предобработка данных и проверены гипотезы. Для приложения чтения использован односторонний t-тест Стьюдента, который не выявил статистически значимой разницы во времени активности пользователей двух городов. В A/B-тесте интернет-магазина проанализирована конверсия в покупку; статистический анализ показал, что новый интерфейс не привёл к целевому росту конверсии. По результатам подготовлены выводы и рекомендации для обеих бизнес-задач.
